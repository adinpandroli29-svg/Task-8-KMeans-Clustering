# Task-8-KMeans-Clustering
AI & ML Internship - Task 8: Clustering using K-Means

📌 Task 8: K-Means Clustering

Objective:
To perform unsupervised learning using K-Means clustering.

Dataset Used:
Iris Dataset (loaded from sklearn.datasets).

Tools Used:

> Python

> Pandas

> Scikit-learn

> Matplotlib

Steps Performed:

> Loaded and scaled dataset

> Applied Elbow Method to determine optimal K

> Implemented K-Means clustering

> Visualized clusters with color coding

>Evaluated clustering using Silhouette Score

Conclusion:
K-Means successfully grouped the dataset into clusters. The Elbow method suggested an optimal number of clusters, and the Silhouette score indicated good cluster separation.
